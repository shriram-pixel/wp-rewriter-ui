WP Rewriter

Find & replace and AI rewrites on a live WordPress site, over SSH.

Connection

Where to connect and which model to use. Settings are saved to config.ini.

Install Claude Code and run claude login once in a terminal. Rewrites then use your Claude subscription — no API key. It’s a bit slower per item than the API.
